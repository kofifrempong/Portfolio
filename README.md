

### Description

A portfolio to showcase my past work as a developer/designers. 


### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap
- Edit Content From One Place

### Setup

Get the code

<pre>git clone </pre>
 
Install required dependencies

<pre>npm install</pre>


Start the server

<pre>npm start</pre>

### More
